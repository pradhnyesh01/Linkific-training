# Car Sales Data Visualization 📊

## Overview

This project performs basic Exploratory Data Analysis (EDA) on a car sales dataset using Python and Matplotlib.
It includes simple visualizations to understand data distribution, relationships, and trends.

## Dataset

* File: `car_sales_data.csv`
* Contains information about car features like engine size, manufacturing year, and brand.

## Features

The project includes the following plots:

* Line Plot
* Bar Plot
* Scatter Plot
* Pie Chart
* Histogram


## Technologies Used

* Python
* Pandas
* Matplotlib
